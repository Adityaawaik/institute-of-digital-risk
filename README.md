# 🏫 Institute of Digital Risk

A responsive web application designed to present information about the **Institute of Digital Risk**, including its mission, programs, and digital risk management initiatives.  
The project focuses on creating a **modern, responsive, and user-friendly interface** for an educational or organizational website.

## 📖 Overview

The **Institute of Digital Risk** website provides information about digital security, risk management awareness, and institutional services.  
The platform is built with modern frontend technologies to ensure a **clean UI, accessibility, and responsiveness across devices**.

## ✨ Features

- Responsive and modern UI  
- Structured sections for institutional information  
- Clean and intuitive navigation  
- Optimized layout for desktop and mobile devices  
- Modular and maintainable code structure
